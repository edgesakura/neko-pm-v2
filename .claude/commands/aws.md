# AWS開発モード

AWS関連の開発・設計をサポートするスキル。

## ナレッジ読み込み

このスキルが呼ばれたら、まず以下を参照：
- `knowledge/aws/` - AWS関連資料

## できること

### 1. インフラ設計
- VPC/サブネット設計
- EKS/ECS構成
- IAMポリシー設計

### 2. CloudFormation / Terraform
- IaCテンプレート作成
- 既存リソースのコード化

### 3. 監視連携
- CloudWatch設定
- Datadog連携設定

### 4. コスト最適化
- リソースサイジング
- Reserved/Savings Plans提案

## 関連アカウント

- cmm: ユーザ管理用AWSアカウント（Datadogと1:1紐付け）

## 使い方例

```
/aws EKSクラスターのノードグループを設計したい
/aws CloudWatchからDatadogにメトリクスを送りたい
/aws IAMポリシーを最小権限で設計したい
```
